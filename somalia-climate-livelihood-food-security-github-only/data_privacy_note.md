# Data Privacy and Use Note

This repository does not contain raw SIHBS 2022 household microdata.

The Somalia Integrated Household Budget Survey 2022 microdata are managed by the Somalia National Bureau of Statistics and are subject to official data-access and data-use conditions.

This repository only shares:

- aggregated summary results;
- derived non-identifiable tables;
- selected publication-style figures;
- reproducible code templates.

No attempt is made to identify individuals, households, or communities. The repository is intended for research communication, transparency of derived outputs, and portfolio demonstration only.
